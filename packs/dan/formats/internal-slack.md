# Format Add-On: Internal Slack

Layer this on top of the master style guide for internal Slack messages and updates.

## Length

2-4 short paragraphs MAX. Anything longer belongs in a doc with a Slack link to it.

## Structure

1. **Lead with the ask or the news.** No "Hey team, hope everyone's having a great week."
2. **Context (one short paragraph).** Only what's needed to act.
3. **Action items.** Use a Slack-friendly checklist:
   - `:white_check_mark:` Done items
   - `:hourglass_flowing_sand:` In-progress items
   - `:red_circle:` Blocked items

## Required elements

- Tag the specific people who need to act. Do not @channel for FYIs.
- If asking for a decision, propose the answer you recommend.
- Link to the source-of-truth doc if one exists.

## Forbidden in Slack format

- "Bumping this" without new information.
- Multi-line code blocks for non-code content.
- More than one emoji in a row.
- Replying in-thread when the original needs editing. Edit the original.
