# Format Add-On: Tweet Thread

Layer this on top of the master style guide for X/Twitter threads.

## Length

5–9 tweets total. Fewer is a one-off; more is unfocused.

## Lead tweet (1/n)

- Must hook in under 240 chars to leave room for engagement.
- Validate the pain point in the first line. Save the resolution for tweet 2.
- No emoji in the lead. No "🧵". The numbering does the job.

## Body tweets

- Each tweet stands alone. A reader who only sees tweet 4 should still get value.
- One idea per tweet. If a tweet has two ideas, split it.
- No screenshots of paragraphs. If it doesn't fit in 280, rewrite it.

## Closing tweet (n/n)

- A direct call to action: link, GitHub repo, demo signup, or "follow for more."
- Do not end with a question begging engagement.

## Forbidden in thread format

- "Here's a thread on…" openers.
- Reusing the same emoji bullet on every tweet.
- Self-replying days later. Finish the thread or don't post it.
- Quote-tweeting your own tweet for "boost."
